# Python
## regex
### syntax

| character | meaning | example |
| ---------- | ---------- | ---------- |
| `.` | matches any character except newline | |
| `^` | matches start of string | |
| `$` | matches end of string | |
| `*` | matches 0 or more times | `ab*` -> 'a', 'ab', or 'ab' with x b's |
| `+` | matches 1 or more times | `ab+` -> 'ab' or 'ab' with x b's |
